##分布式计算之storm

###1. 安装部署

  • 环境：10.77.100.68
    ####1.1maven安装
  • 下载地址：https://maven.apache.org/download.cgi
  • 安装说明：https://maven.apache.org/install.html
  • 操作命令：
    • wget http://mirrors.hust.edu.cn/apache/maven/maven-3/3.5.4/binaries/apache-maven-3.5.4-bin.tar.gz
    • tar -xzvf apache-maven-3.5.4-bin.tar.gz
    • mv apache-maven-3.5.4 /opt/
    • export PATH=/opt/apache-maven-3.5.4/bin:$PATH

####1.2 storm-docker镜像

  • 镜像: https://hub.docker.com/\_/storm/
  • docker pull storm:latest
  • docker run -it -v storm-starter-topologies-1.0.2.jar:/topology.jar storm:1.0.2 storm jar /topology.jar org.apache.storm.starter.ExclamationTopology
    *

####1.3 storm - 下载

  • 示例参考：examples/storm-starter/README.markdown
  • sourcecode: git://github.com/apache/storm.git
  • package: wget http://mirror.bit.edu.cn/apache/storm/apache-storm-1.2.2/apache-storm-1.2.2.tar.gz

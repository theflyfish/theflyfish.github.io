apiserver 登录方式一

  • 登录master节点，开启代理
  • 获取本机IP：

    /sbin/ifconfig -a|grep -A 2 eth0|grep inet|grep -v 127.0.0.1|grep -v inet6|awk ‘{print $2}’|tr -d “addr:”

  • 开启代理

    kubectl proxy –port=8001 –address=’10.85.137.70’ –accept-hosts=’^.*’

  • 获取登录token

    kubectl describe serviceaccount tiller -n kube-system
    kubectl describe secret tiller-token-6w8tt -n kube-system

apiserver登录方式二
在个人电脑上配置kubectl，配置证书，本地起代理，浏览器访问本地代理

  1. 配置准备
  • python -m SimpleHTTPServer 9999
  • 从master拉取配置：~/.kube/config
  • 从master拉取证书：/etc/kubernetes/pki
  • 从master中拷贝客户端kubectl
  1. 导入证书：
  • mac: lanchpad->实用工具->钥匙串访问->导入项目：
  • 将pki中的apiserver.crt与ca.crt 证书导入
  • 重启浏览器
  1. 启动本地代理：
  • ./kubectl proxy –port=8001 –address=’127.0.0.1’ –accept-hosts=’^.*’ –kubeconfig=admin.conf

  1. 获取token，访问dashbroad 同方法1

  2. 访问127.0.0.1:8001
